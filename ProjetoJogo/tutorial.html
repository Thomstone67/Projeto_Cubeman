Bem Vindo

Cubeman é um jogo estilo plataforma, isso significa que você usará os direcionais no teclado para movimento




Algumas plataformas te dão um boost de pulo ao atravessa-las de baixo para cima


Ao cair de uma plataforma sem pular antes de dará um pulo extra enquanto estiver no ar para uma aterrissagem mais controlada


Tente pular por baixo de uma plataforma para outra para um boost vertical


Ao atravessar os limites da tela horizontalmente o cubo será trazido para o outro lado da tela, use isso para resolver os puzzles


As bolas azuis irão revelar uma parede ou plataform ilusória que estará entre você e o objetivo


As bolas cinza são o seu objetivo
